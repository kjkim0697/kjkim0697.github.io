소개

전기차(EV) 및 에너지 시스템 분야에서 강화학습(RL)을 활용한 제어 알고리즘을 연구·개발하고 있습니다. 특히 초고속 충전 제어, 시스템 최적화, 데이터 기반 의사결정 문제에 관심이 많습니다.

전문 분야

  • ISO SC22 전기자동차 배터리, 탄소발자국 표준 의장 (ISO 25027)
  • 강화학습 / 딥강화학습 (DQN, PPO, SAC 등)
  • 전기차 충전 제어 알고리즘
  • EV 파워트레인 및 에너지 관리
  • Python, PyTorch, MATLAB

주요 프로젝트

강화학습 기반 EV 초고속 충전 제어

배터리 열화 최소화와 충전 시간 단축을 목표로 한 강화학습 기반 충전 제어 알고리즘 개발. 기존 rule-based 제어 대비 성능 향상을 달성.

EV 충전 데이터 기반 최적화 연구
